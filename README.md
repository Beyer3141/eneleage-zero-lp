# ENELEAGE Zero - 電気代削減LP

## 概要
太陽光パネル不要のAI蓄電池ENELEAGE Zeroのランディングページと料金シミュレーター

## 機能
- レスポンシブデザイン対応
- 月別電気代推移シミュレーター
- 実データに基づく削減額計算
- 問い合わせフォーム統合

## 開発
```bash
# 開発サーバー起動
pnpm dev

# ビルド
pnpm build

# 本番環境起動
pnpm start
```

## 技術スタック
- Next.js 16
- TypeScript
- Tailwind CSS
- Recharts
- Radix UI
